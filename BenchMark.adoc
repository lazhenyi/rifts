= Rifts Benchmark Results

Results from the Criterion-based benchmark suite covering the full `rifts` crate
surface. Generated automatically via `examples/update_benchmark_readme.rs`.

== Running benchmarks

[source,sh]
----
# Build the entire suite
cargo bench --no-run --all-features

# Run stable benchmarks (no network, no disk)
cargo bench

# Feature-gated benchmarks
cargo bench --features sled --bench sled_store
cargo bench --features client --bench websocket_loop

# Throughput benchmarks (ops/s via Throughput::Elements)
cargo bench --bench throughput

# Update this file after running benchmarks
cargo run --example update_benchmark_readme -- BenchMark.adoc
